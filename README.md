 מערכת ניהול משתמשים - Spring Identity Core

תיאור הפרויקט: מערכת Backend מבוססת Java Spring Boot לניהול הרשאות וכניסת משתמשים. המערכת תומכת בשני סוגי תפקידים: מנהל (Admin) ומשתמש (User), עם הפרדת גישה לדפים ומשאבים שונים.

טכנולוגיות: Java 17+, Spring Boot, PostgreSQL, Docker & Docker Compose
